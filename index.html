Apparticle

A Mac OS X particle editor for and with Cocos2D

Download this project as a .zip file

About

Apparticle is an open source GPLv3 Mac OS X 10.8 software to edit particle systems and save them in the .plist format to be used in Cocos2d based project. Apparticle is still in alpha.
